CodeTreeViewer

사용자가 선택한 프로젝트 내부의 파일들간에 관계도를 태양계 나타내듯이 그리거나 트리 형태로 그리는 등 시각적으로 나타내느 프로젝트.


프로젝트 시각적 표시 기능

* 프로그램에서 제공하는 시각적 효과는 모두 관계도, 트리 형태 보여줘야 함
* 사용자가 직접 컴퓨터 내의 프로젝트 폴더 선택
* 여러개의 탭이 존재하여 각 탭마다 다른 프로젝트 선택 가능
* 관계도 <-> 트리 형태 전환 가능
* 특정 파일을 클릭하면 해당 파일을 중심으로 부모, 자식 관계 시각적으로 표시
* 특정 함수를 클릭하면 해당 함수를 마지막 노드로 호출되는 곳들을 시각적으로 표시
* 파일, 함수 검색 기능
* 색을 구분지어 프로젝트의 영역 구분해서 표시(예를 들어 안드로이드에서 Application 영역은 주황색, Framework 영역은 초록색, Stack 영역은 파란색으로 표시)


로그 덤프 파일 분석 기능

* 로그들이 저장된 덤프 파일을 열어볼 수 있는 영역 제공
* 특정 로그 라인을 선택하면 해당 로그 라인을 입력하는 부분을 프로젝트에서 탐색하여 표시
* 역시 해당 파일 혹은 함수의 관계를 시각적으로 함께 표시
* 로그 검색 기능


프로젝트 환경 구성

* 브라우저에서 띄우는 형태가 아닌 윈도우, 우분투에서 동작할 별도의 실행 프로그램이어야 함
* 최소화, 최대화, 종료 기능 포함해야함
* 프로그램이 종료되어도 항상 마지막에 띄웠던 프로젝트, 팡리이 유지되어야함
